--- calendar | Calendar Management
  • Multiple calendars — Create separate calendars for work, personal, etc.
  • Color coding — Assign colors to calendars to distinguish events at a glance.
  • Visibility — Toggle calendar checkboxes in the sidebar to show/hide their events.
  • Subscribed calendars — Calendars shared with you appear under "Other calendars".
--- calendar-plus | Events
  • Create — Click a date or drag a time range on the calendar, or press N.
  • Quick duration — Use duration chips (30m, 1h, 2h...) when creating an event.
  • Invite guests — Search and add users as guests to your events.
  • Respond — Accept or decline event invitations from the detail panel.
  • Detail panel — Click an event to open its details in the right panel.
--- code | API Access
  • Base URL: /api/v1/calendar
  • Authentication: Use your session cookie or API token.
  • Documentation: Full API docs are available at /schema/swagger-ui/.
