SyntaxResult
**bold**bold
*italic*italic

Conversations
  • Direct messages — Start a private conversation with one person.
  • Group chats — Add two or more people and optionally name the group.
  • Reactions — Hover over a message and click an emoji to react.
  • Edit & delete — Hover over your own messages to edit or delete them.
  • Search — Use the command palette (Ctrl+K) to quickly find and switch conversations.
API Access
  • Base URL: /api/v1/chat
  • Authentication: Use your session cookie or API token.
  • Documentation: Full API docs are available at /api/docs.
