{% comment %} Refresh button that triggers an Alpine AJAX swap on a target element. Required context: - url_expr: Alpine expression evaluated for the href (e.g. "feedUrl"). - target: id(s) of the element(s) to swap. Passed as-is to x-target. Optional context: - loading_expr: Alpine expression (default "loading") that, when truthy, spins the icon. - title: tooltip text (default "Refresh"). - size: button size modifier (default "sm"). Note: the reactive ``:class`` binding lives on a ```` wrapper rather than the ```` itself. Lucide replaces ```` with a fresh ```` - the spin would never start. Wrapping in a ```` (which Lucide never touches) preserves the binding and makes the SVG inside rotate via ``animate-spin``. {% endcomment %}
