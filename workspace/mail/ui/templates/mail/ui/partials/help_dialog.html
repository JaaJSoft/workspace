--- mail | Mail Features
  • Multiple accounts — Add and manage several mail accounts from one place.
  • Compose — Write new messages with To, Cc, and Bcc fields. Contact autocomplete suggests addresses from your history.
  • Drafts — Drafts are saved automatically and can be restored when reopening compose.
  • Attachments — Attach files when composing and download attachments from received messages.
  • Search & filters — Search messages by subject, snippet or sender. Filter by unread, starred, or with attachments.
  • Batch actions — Select multiple messages to mark read/unread, star, or delete in bulk.
  • Folders — Right-click a folder for options: refresh, mark all read, change icon, or sync.
--- code | API Access
  • Base URL: /api/v1/mail
  • Authentication: Use your session cookie or API token.
  • Documentation: Full API docs are available at /schema/swagger-ui/.
