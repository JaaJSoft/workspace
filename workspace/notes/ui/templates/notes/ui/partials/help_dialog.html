Keyboard Shortcuts

Notes
  • Create — Click the + button in the note list panel to create a new note.
  • Edit — Select a note to open the Markdown editor with live preview.
  • Rename — Click the note title in the editor toolbar to rename it.
  • Favorite — Click the star icon to mark a note as favorite.
  • Tags — Use the tag manager to organize notes with labels.
  • Delete — Click the trash icon in the toolbar (confirmation can be toggled in preferences).
Folders & Organization
  • Folders — Organize notes in a nested folder tree (up to 4 levels deep).
  • Group folders — Create shared folders linked to your groups.
  • Quick access — Use All, Favorites, Recent, and Journal views in the sidebar.
  • Context menu — Right-click folders and tags for quick actions.
Journal
  • Daily notes — The journal automatically creates a note for today's date.
  • Auto-folder — A dedicated "Journal" folder is created the first time you use it.
  • Toggle — Show or hide the Journal entry in preferences.
